  • 不询问关闭窗口 
    window.opener = null;
window.close();

tr.rowIndex

位于字符串加法拼接的表达式中，三元符号需要被括在括号中，否则下面的表达式报错。 

var a = null;
"" + a ? a.toString() : ""

需要： 

var a = null;
"" + (a ? a.toString() : "")
